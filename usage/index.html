    Usage Guide¶

    This usage guide provides step-by-step instructions for getting up and running with InfDB.

    Quick Start Overview¶

    To get started with InfDB, you will generally follow this path:

    1. Check Requirements: Ensure your system has the necessary prerequisites.
    2. Get Software: Clone the repository.
    3. Setup: Configure services and parameters.
    4. Deployment: Launch the InfDB stack.
    5. Services: Description of InfDB microservices.
    6. Tools: Software that processes data and is connected to InfDB.
    7. Demo: Example applications using InfDB tools.

    Select a topic from the navigation menu to begin.
