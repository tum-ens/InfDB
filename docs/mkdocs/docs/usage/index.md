# Usage Guide

This usage guide provides step-by-step instructions for getting up and running with **InfDB**.

## Quick Start Overview

To get started with InfDB, you will generally follow this path:

1.  [**Check Requirements**](requirements.md): Ensure your system has the necessary prerequisites.
2.  [**Get Software**](get-software.md): Clone the repository.
3.  [**Setup**](setup/index.md): Configure services and parameters.
4.  [**Deployment**](deployment.md): Launch the InfDB stack.
5.  [**Services**](services.md): Description of InfDB microservices.
6.  [**Tools**](tools.md): Software that processes data and is connected to InfDB 

Select a topic from the navigation menu to begin.
